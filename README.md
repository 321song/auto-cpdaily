# 今日校园自动签到-云函数版

### 0、版本

此脚本分为两个版本

现在浏览的版本是云函数版：云函数版可以通过将文件上传到腾讯云函数，定时执行，不需要有服务器，也是基本免费的。

另一个是普通版：[链接](https://github.com/aowubulao/auto-cpdaily/tree/master)，普通版需要程序一直在电脑的后台运行，开箱即用，适合有服务器的人使用。



### 1、使用

下载：[下载链接1](http://106.14.251.237/serverless.zip)
